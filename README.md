# B&Brother’s - Landing Page (GitHub Pages Ready)

Questa è la landing page di B&Brother’s ottimizzata per GitHub Pages.

## Come pubblicare

1. Vai su [github.com](https://github.com) e crea un nuovo repository pubblico, ad esempio `bbrothersrome`.
2. Carica i file di questa cartella (`index.html`, `brochure.pdf`, `robots.txt`, `favicon.ico`, `README.md`).
3. Vai su **Settings → Pages**.
4. In **Branch**, seleziona `main` e cartella `/ (root)`.
5. Salva: in pochi minuti il sito sarà disponibile all'indirizzo:

```
https://TUONOME.github.io/bbrothersrome/
```

Sostituisci `TUONOME` con il tuo username GitHub.
